Instrucciones de Uso

Esta aplicacion es utilizada por Magneto para reconocer el Adn de las personas, tanto humanas como mutantes, para asi, seleccionar a los mutantes para que pertenezcan a su equipo.

Mediante esta aplicacion, se debera enviar una cadena de ADN, conformada por [NxN], osea, la cadena debe tener tantos elementos como caracteres tenga cada uno de sus elementos; por ejemplo la Cadena: "AAAA" "CCCC" "TTTT" "GGGG", para su comodidad, le damos la opcion de separar cada fragmento de la cadena con espacios, puntos o comas.

Sera aceptada como una cadena valida, la cual sera posible analizar para ver si es ADN Humano o Mutante, en caso que la cadena sea Valida, vera una notificacion indicando que se guardo correctamente el dato, sino un mensaje de error, con una sugerencia para que vuelva a intentarlo.

Se considera a un mutante si tiene una secuencia de por lo menos cuatro bases nitrogenadas continuas, ya sea horizontalmente, verticalmente y oblicuo. Para esta validacion se distribuye la cadena de Adn ingresada en una matriz de NxN y se realiza dicha verificacion.

Nota: Las cadenas de ADN solo pueden estar conformadas por las bases nitrogenadas A C G T y por lo menos deben ser de [4x4] para poder comprobar si son Mutantes o Humanas, por lo que si en la cadena se encuentra alguna otra base nitrogenada, o la cadena ingresada es menor a [4x4] sera invalida y tambien se devolvera error.

Se podra ingresar a la solapa de estadisticas para verificar cuales de las muestras analizadas corresponden a Mutantes, cuales a Humanos y un Ratio entre ellas. Asi mismo se podra seleccionar un mail en el cual desee recibir estas estadisticas.
